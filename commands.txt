git --version :to check the git version installed
git config list: To show configuration of the list created .
git config --global user.name 'username' : to add user name
git config --global user.email 'email' : to add email
git init: to initialize git
git status :to check the status
git add -A : to add modifications from local working directory to staging area.
git commit -m "your message" : to commit the changes to local repository
git log:to show modification with dates of initialisation
